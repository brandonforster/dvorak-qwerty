Dverty

Dverty is a regular US Dvorak keyboard layout but toggles to Qwerty while the Ctrl button is held down.

Keyboard shortcuts are always designed with Qwerty layouts in mind. Undo, Cut, Copy, Paste can all be done in Qwerty with one hand, but not in Dvorak. Dverty solves this problem.

Dverty has only been tested on a Windows XP system. Use at your own risk.

Also available is a modified version of DVAssist, a neat program that lets you easily switch between Qwerty and Dvorak. This version switches between Qwerty and Dverty.

Notes

There are known issues where some keys in CapsLock mode are broken. I haven't had time to fix this.

Check out the dvorak-qwerty project for a solution that works on Linux and Windows. I haven't tried this yet, though it looks promising.

    • Installation

    (64 bit note: use kbdverty64.dll and dverty64.reg)

    1. Copy 'kbdverty.dll' to 'C:\WINDOWS\system32\'
    2. Execute 'dverty.reg' to add the necessary keys to the Registry
    3. Reboot Windows
    4. Add the new layout through the Control Panel
      1. Open the Windows Control Panel
      2. Select 'Regional and Language Options'
      3. Select the 'Languages' tab
      4. Click 'Details...'
      5. Click 'Add...'
      6. Select 'United States-Dvorak (Ctrl + Qwerty)'
      7. Click the s#!t out of 'OK' until everything goes away
